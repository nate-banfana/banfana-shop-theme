# Banfana Shopify Theme

Custom Online Store 2.0 theme for the Banfana demo store (shop.banfana.com).

- Dark + gold branding, real BANFANA wordmark, self-aware demo copy
- Full sections/schema (hero, value props, collection list, featured products, reviews carousel, blog, newsletter, footer with policy links)
- Native Okendo-style reviews (assets/reviews.js) with aggregateRating JSON-LD
- SEO: keyword-rich titles, meta descriptions, Product + BlogPosting structured data
- Blog list + article templates

## GitHub ↔ Shopify sync
Connected via Shopify's GitHub integration (Online Store → Themes → Add theme → Connect from GitHub).
Push to the connected branch → the live theme updates automatically. No zip uploads.

Theme files live at the repo root (assets/, config/, layout/, locales/, sections/, snippets/, templates/).
